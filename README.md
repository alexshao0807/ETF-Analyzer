# 📈 ETF Portfolio Analyzer (ETF 持股變動比對系統)

**[📥 下載 Releases](https://github.com/alexshao0807/ETF-Analyzer/releases) | [🛠️ 技術文檔](#-技術堆疊-tech-stack) | [📖 使用說明](#-使用說明-usage) | [📝 開發紀錄](#-開發紀錄-release-notes) | [👤 作者聯繫](#-作者-author)**

`ETF Portfolio Analyzer` 是一個專為量化投資人與分析師設計的 Windows 桌面工具。透過高效能比對演算法，在秒級內精確識別 ETF 成份股權重與持股數量的細微變化，解決人工處理 Excel 耗時且易錯的痛點。

> **開發初衷**：在投資市場中，數據的即時性決定了決策的品質。我們將複雜的 Excel 比對邏輯封裝，讓分析師能專注於變動背後的投資邏輯，而非繁瑣的資料清洗。

---

## 💡 投資分析哲學

> 「真正的預測，就是不測而測。所有預測的基础，就是分類與數據的精確比對。」

傳統的 ETF 觀察僅停留在淨值波動，但成分股的**權重調整**與**股數異動**往往預示了法人（如投信、外資）的調倉方向。本工具透過 $O(1)$ 的 Hash 比對演算法，確保在處理數萬筆持股資料時依然保持極速響應，將市場變動分段化，讓投資者能迅速確定邊界條件並做出反應。

---

## ✨ 核心功能 (Key Features)

* **🚀 高效能比對演算法 ($O(1)$)**：採用 Hash-based 資料結構 (Dictionary) 優化搜尋邏輯，即使處理數萬筆成分股資料也能瞬間完成，效能大幅優於傳統巢狀迴圈 ($O(N^2)$)。
* **⚡ 非同步多執行緒 (Async/Await)**：導入非同步設計模式，確保在讀取大型 Excel
